# 📹 Registros Audiovisuales y Evolución

Este directorio recopila la evidencia multimedia del proyecto "Equili-beat", documentando empíricamente el progreso desde las pruebas iniciales de hardware hasta el equilibrio dinámico autónomo.

### 🎬 Evolución Experimental (Videos)
El desarrollo se dividió en 3 etapas clave. Se adjunta un video (`.mp4`) representativo de cada fase:

1. **Avance 1 - Unidad de Motores:** Pruebas iniciales de tracción, validando la respuesta de los motores DC y el driver L298N ante comandos básicos de movimiento.
2. **Avance 2 - Integración de Sensores:** Pruebas preliminares vinculando la lectura del sensor inercial (MPU6050) con el accionamiento de los motores.
3. **Avance 3 - PID y Estética Final:** Demostración del robot operando de forma autónoma. Se observa el sistema de control PID sintonizado manteniendo el equilibrio estable frente a perturbaciones, incorporando además la estética vintage terminada.

### 🛠️ Nota sobre la Evolución Estructural
Al revisar los registros, se evidenciará que **el chasis y la disposición física del robot son diferentes en cada video**. La estructura mecánica sufrió modificaciones iterativas en cada avance para solucionar problemas físicos encontrados en el camino (como falta de rigidez, vibraciones parásitas, mala distribución del centro de masa y fallos en las sujeciones), hasta llegar a la versión final optimizada con impresión 3D y corte láser.
