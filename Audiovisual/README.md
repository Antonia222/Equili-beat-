# 📹 Registros Audiovisuales

Este directorio recopila la evidencia experimental, pruebas de ensayo/error y el resultado final del proyecto "Equili-beat".

### Contenido Multimedia:
1. **Fotografía del Prototipo Final:** Captura del ensamblaje definitivo destacando la estética vintage/rústica inspirada en un tocadiscos clásico.
2. **Videos Experimentales (`.mp4`):** * Pruebas iniciales demostrando el efecto de saturación y vibración (Windup) al utilizar los valores teóricos puros de Ziegler-Nichols.
   * Pruebas finales demostrando el **equilibrio dinámico estable** logrado tras la sintonización empírica del controlador PID (reducción de la constante derivativa Kd a 0.7 para atenuar el ruido mecánico).
